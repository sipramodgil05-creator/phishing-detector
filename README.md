# Phishing Email Detection System

This project is a Machine Learning-based phishing email detector. It uses Natural Language Processing (NLP) and machine learning techniques to classify emails as phishing or legitimate. The project is built using Python, Scikit-learn, and Streamlit for real-time classification via a user-friendly web interface.

## Features
- Real-time phishing email classification using Logistic Regression and TF-IDF vectorization.
- Phishing risk scoring based on keyword matching and email characteristics.
- Web interface built with Streamlit for easy interaction.
- Explainability: The model provides insights into why a certain email was classified as phishing.

## Technologies Used
- Python: Primary programming language
- Scikit-learn: For machine learning and vectorization (TF-IDF, Logistic Regression)
- Streamlit: For building the web app and real-time email classification
- NLP: Natural Language Processing for extracting features from email text
- Pandas: For data manipulation
